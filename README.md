# Contact_List

Full-stack CRUD web application for managing a list of contacts.  
Users can **create**, **read**, **update**, and **delete** contacts through a modern React frontend backed by a Flask REST API.

# Backend Setup (Flask)

1. Go to backend folder
   
cd backend

2. Install Python dependencies

pip install -r requirements.txt

3. Run the backend server

run main.py

# Frontend Setup (React)

1. Go to frontend folder

cd frontend

2. Install dependencies
   
npm install

3. Start the development server
   
npm run dev
